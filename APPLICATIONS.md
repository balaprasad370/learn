# Applications log

Update every time you apply or hear back. Keeps the pipeline honest.

| Date | Company | Tier | Channel | Role | Comp band (known) | Status | Next step | Notes |
|------|---------|------|---------|------|-------------------|--------|-----------|-------|
|      |         |      |         |      |                   |        |           |       |

## Status values

- `applied` — submitted, no response yet
- `screening` — recruiter call scheduled/done
- `tech-1`, `tech-2`, `tech-3` — technical rounds
- `onsite` — final loop
- `offer` — offer received
- `negotiating` — counter in progress
- `signed` — accepted
- `rejected` — by them
- `withdrew` — by you
- `ghosted` — no reply after 2 weeks

## Channel values

- `direct` — careers page
- `referral:<name>` — referred by a person
- `recruiter-inbound` — they reached out
- `linkedin-easy` — LinkedIn Easy Apply
- `wellfound` — wellfound.com
- `hn-whoishiring` — Hacker News monthly thread
- `turing`, `toptal`, etc. — aggregators
