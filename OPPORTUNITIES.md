## SECTION 1 — Uplers (platform.uplers.com)

**What it is:** Indian talent platform that places vetted engineers into global remote full-time roles. You create a profile, they match you to clients, interviews happen in days.

- **Pay model:** Most full-time placements start at ~$2,500/month base (~25 LPA) for mid-level, rising with seniority. Senior SWE (8+ YoE) roles posted at up to **₹5,00,000/month (~60 LPA)**. Realistic senior band: ~30–60 LPA depending on role + client.
- **Fit:** Good as a parallel pipeline, not a primary one. You don't directly see clients, they control the match.
- **How to apply:** platform.uplers.com → create profile → take AI-interview → wait for matches.
- **Sample open roles (as of Apr 2026):**
  - Senior DevOps Engineer (remote) — opportunity ID HR1177549578129451
  - Senior Software Engineer (remote) — opportunity ID HR1175994137981328
  - ~23 open positions listed under Sarvam AI via Uplers (partnered placement)

**Verdict:** Apply, but don't over-rely. Treat as a stream, not the river.

## SECTION 4 — Generic JD pattern for senior SWE roles (5 YoE, ₹45–65 LPA)

Every listing I pulled converges on the same shape. Your resume should hit all of these bullets:

**Must-have:**
- 5+ years in a production codebase, shipping weekly or faster.
- Ownership of at least one complex feature/system end-to-end (design → ship → on-call).
- Strong in one backend language (Python / Go / Node / Java / Rust) and one frontend framework (React / Next).
- Cloud + container fluency: AWS or GCP, Docker, Kubernetes.
- System design interviews: URL shortener, rate limiter, feed ranking, chat, payments.
- For AI roles: LLM APIs, RAG, vector DBs, fine-tuning basics, at least one shipped AI feature.

**Nice-to-have (AI/ML premium):**
- LangChain / LlamaIndex / custom agent orchestration.
- Vector DB (Pinecone, pgvector, Weaviate).
- Fine-tuning / LoRA / quantization.
- Built something 0→1 with real users.

**Red-flag filter:**
- Cash < 40 LPA at senior level → skip unless equity is 10× normal.
- No comp band shared after 2 recruiter asks → skip.
- "Up to X LPA" phrasing where X is really the staff/principal ceiling.
